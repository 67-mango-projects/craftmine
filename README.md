# CraftMine
A game created on Opengl and C++ for school project

## Keybinds
| Button | Action |
| :---: | :------: |
| W A S D | Movement | 
| Tab | View Inventory |
| Left Click | Break/Attack | 

## Features
- Procedural World Generation
- Inventory
- Crafting
- Projectiles
- Bosses
- Dimensions
